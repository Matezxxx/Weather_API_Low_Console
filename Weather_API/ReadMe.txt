# WeatherConsoleApp
Jedná se o jednoduchou konzolovou aplikaci k získání dat o počasí pomocí OpenWeather API.
Přidán jednoduchý do-while cyklus pro možnost opakovaného vyhledávání dat o počasí v různých městech
## Funkce
- Zobrazení aktuální teploty, vlhkosti, rychlosti větru a tlaku ve zvoleném městě.
- Data získána z [OpenWeather API](https://openweathermap.org/).

## Použití
1. Zadejte název města do konzole.
2. Aplikace zobrazí informace o počasí.
3. Pro pokračování aplikace stiskněte "y" a potvrďte entrem. Pro ukončení aplikace stiskněte dvakrát libovolnou klávesu.

## Jak nastavit
1. Získejte API klíč z [OpenWeather API](https://openweathermap.org/).
2. Vložte klíč do appsettings.json.

This is a simple console application for retrieving weather data using the OpenWeather API.
A simple do-while loop has been added to allow repeated searches for weather data in different cities.

## Features
Displays the current temperature, humidity, wind speed, and pressure for the selected city.
Data is retrieved from OpenWeather API.

## Usage
Enter the name of the city into the console.
The application will display the weather information.
To continue using the application, press "y" and hit Enter. To exit the application, press any key twice.

## Setup
Obtain an API key from OpenWeather API.
Insert the key into appsettings.json.
